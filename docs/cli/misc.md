# `agentx providers` / `agentx version`

## `agentx providers`

List supported LLM providers and the env vars each needs.

```bash
agentx providers
```

See the full table in [Providers](../providers.md).

## `agentx version`

Print the installed version.

```bash
agentx version
```

```bash
agentx version && agentx providers   # quick sanity check after install
```
