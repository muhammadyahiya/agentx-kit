# agentx-kit v1.1.0 — Release Notes

**Git tag:** `v1.1.0`

---

## Highlights

- **Flow viewer edit-in-place.** `agentx flow --serve`'s DAG side panel now has an **✎ Edit**
  button. Clicking it swaps the read-only source view for a live [Monaco](https://microsoft.github.io/monaco-editor/)
  editor; **Save** writes the change straight back to the source file on disk, overwriting
  *exactly* that function/class's lines (computed from the AST's `end_lineno`) — nothing else in
  the file is touched. See [Flow — code as a DAG](../features/flow-dag.md#edit-in-place).
- **New `/api/save` endpoint** on the `--serve` local server, token-gated like every other route.
- **`end_lineno`** is now part of every flow node's payload (`htmlgen.py`), enabling precise
  span-replacement edits.
- **Full documentation site** published to GitHub Pages:
  <https://muhammadyahiya.github.io/agentx-kit/> — every CLI command and feature documented with
  copy-pasteable examples, plus a detailed [Python API usage guide](../library-usage.md).

## Upgrading

```bash
pip install -U agentx-kit
```

No breaking changes — this is an additive release.
